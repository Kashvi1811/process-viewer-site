## ✨ Overview
Process State Visualizer is an interactive, single-page educational web app that demonstrates how operating-system processes move through these core states:

🆕 NEW → 🟨 READY → 🟩 RUNNING → 🟥 WAITING → ⛔ TERMINATED

It combines lifecycle animation, scheduler behavior, CPU timelines, and process analytics in one concise and professional interface.

## 🚀 Feature Highlights
- 🧠 Lifecycle Board: visual state lanes for each process state
- 🎮 Simulation Controls: New Process, Step Forward, Auto Run, Clear All
- ⚙️ Scheduler Modes: Round Robin and FCFS with configurable quantum
- 🧪 Custom Process Editor: presets + JSON workflow builder
- 📈 CPU Timeline: per-process transition slices with time-step context
- 📝 Activity Log: live transition history with readable timestamps
- 📊 Process Statistics: overall + per-process WT, RT, TAT, and executed slices
- 👀 Simple Mode: distraction-free view for demos and quick teaching

## 📊 Metrics Included
- ⏳ Average Waiting Time
- 🔁 Average Turnaround Time
- ⚡ Average Response Time
- 🧮 CPU Utilization
- 🚦 Throughput
- 🧷 Per-process execution and timing breakdown

## ▶️ Quick Start
1. Open this folder in VS Code.
2. Launch [index.html](index.html) in your browser (or use Live Server).
3. Create a process and run transitions manually or automatically.

✅ No build step, package install, or backend required.

## ⌨️ Keyboard Shortcuts
- `N` → Create new process
- `Right Arrow` → Step forward
- `Space` → Toggle auto run
- `Esc` → Close custom editor modal

## 🌐 GitHub Pages
